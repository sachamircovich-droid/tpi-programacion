## Descripción

Sistema de gestión de países desarrollado en Python utilizando archivos CSV.

## Funcionalidades

- Agregar país
- Actualizar datos
- Buscar países
- Filtrar paises
- Ordenar paises
- Mostrar estadísticas

## Instrucciones de Uso
- Tener instalado **Python 3.x**.
- Contar con el archivo 'paises.csv' en la misma carpeta que el script ejecutable.

### Ejecución
Para iniciar el programa, abre tu terminal en el directorio del proyecto y ejecuta:

python TPI_1.py
